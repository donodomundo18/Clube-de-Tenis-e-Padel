# Clube de Tenis e Padel

Web App de gestao de reservas de campos desportivos de um clube de tenis e padel de pequeno porte.

## Descricao

Aplicacao web para a gestao integral de um clube desportivo. Permite que atletas pesquisem e reservem campos, gerindo as suas reservas mediante autenticacao. A equipa do clube (gestor e rececionista) pode administrar campos, atletas, reservas, registar pagamentos e efetuar check-in.

## Funcionalidades

- Registo e login de atletas com password encriptada
- Pesquisa e reserva de campos por tipo (Padel Coberto, Padel Descoberto, Tenis Terra Batida, Tenis Rapido)
- Suplementos de iluminacao noturna e aluguer de material
- Gestao de reservas pelo atleta (editar e cancelar ate 24h antes)
- Backoffice com dois perfis: gestor e rececionista
- Registo de pagamentos simulados (parcial/total) e check-in
- Relatorios de ocupacao e receita

## Tecnologias

- HTML e CSS
- PHP
- MySQL
- Apache (MAMP)
- JavaScript

## Instalacao

1. Copiar a pasta Clube para htdocs do MAMP
2. Importar o ficheiro database.sql no phpMyAdmin
3. Ligar o MAMP (Apache + MySQL)
4. Abrir o browser em http://localhost/Clube
